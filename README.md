# From Data to Drama: Visualizing Hollywood’s Most Profitable Stories 🎬📊

This project explores the *Hollywood’s Most Profitable Stories* dataset using **Tableau** to uncover insights about films, studios, genres, and their profitability.

## 📂 Dataset Overview
The dataset includes:
- **74 films**
- **13 lead studios**
- **6 genres**
- Key metrics: *Audience Score, Rotten Tomatoes %, Profitability, Worldwide Gross*

## 📊 Dashboard Features
The interactive Tableau dashboard visualizes:
- **Films by Year, Genre, and Studio**
- **Audience vs. Critic Score Comparisons**
- **Worldwide Gross Distribution**
- **Detailed Film Info with Profitability Insights**
- **Studio–Genre Profitability Mapping**

## 🔎 Key Insights
- Disney’s **Animation films** (e.g., *WALL-E, Tangled, Gnomeo and Juliet*) performed strongly in both **revenue** and **ratings**.
- Clear trends emerge when comparing **audience scores vs. critic scores**, highlighting differences in perception.
- Certain genres consistently drive higher profitability compared to others.

## 🖼️ Dashboard Views
- **Full Dataset View** – high-level exploration across all films.
- **Filtered Animation Genre View** – detailed look at Disney’s standout performance.
